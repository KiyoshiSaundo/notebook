# Типы User Fields

- `enumeration` - Список
- `double` - Число
- `integer` - Целое число
- `boolean` - Да/Нет
- `string` - Строка
- `file` - Файл
- `video` - Видео
- `datetime` - Дата/Время
- `iblock_section` - Привязка к разделам инф. блоков
- `iblock_element` - Привязка к элементам инф. блоков
- `string_formatted` - Шаблон
- `crm` - Привязка к элементам CRM
- `crm_status` - Привязка к справочникам CRM
- `date` - дата
